Blinkit Sales Performance Dashboard | Power BI
 Project Overview

This project focuses on analyzing sales performance data for Blinkit using Power BI.
The dashboard provides a comprehensive view of sales trends, outlet performance, and product category contribution, enabling stakeholders to make data-driven business decisions through interactive visuals.

The project demonstrates practical use of Power BI, DAX, and data visualization best practices.

 Objectives

Analyze overall sales performance and order trends

Compare performance across different outlet tiers

Identify high-performing product categories

Enable interactive analysis using slicers and tooltips

Present insights in a clean, business-ready dashboard

 Dataset Description

The dataset contains historical sales and order-level information with the following key fields:

Order_ID – Unique identifier for each order

Order_Date – Date of the transaction

Sales – Total sales value

Item_Type – Product category

Outlet_Tier – Outlet classification (Tier 1, Tier 2, Tier 3)

Outlet_Size – Size of the outlet

Rating – Customer rating

 Note: The dataset used is for analytical and demonstration purposes.

 Key KPIs

The dashboard tracks the following metrics using DAX measures:

Total Sales

Total Orders

Average Order Value

Average Rating

Orders per Outlet

 Dashboard Features

KPI Cards for quick performance overview

Yearly Sales Trend using an area chart

Outlet Tier Analysis with column charts

Product Category Contribution using a donut chart

Interactive Slicers (Outlet Tier, Item Type, Year)

Report Page Tooltips for deeper insights without clutter

Consistent color theme and clean layout for business readability

 Tools & Technologies

Power BI Desktop

DAX (Data Analysis Expressions)

Data Modeling

Interactive Visualizations

 Key Insights

Tier 1 outlets generate the highest order volume

Sales show consistent growth with peak performance in recent years

Household and Frozen Food categories contribute significantly to revenue

Interactive filtering enables quick drill-down analysis by outlet and product
